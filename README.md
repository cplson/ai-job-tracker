# AI Job Tracker

A full-stack application for tracking job applications and improving application materials using AI.

Tech stack:
- .NET Web API
- React
- PostgreSQL
- Redis
- Docker
- OpenAI API

This project is being built as a capstone project for a web programming degree.

## Demo / presentation data

From `backend/`, after the database is initialized:

```bash
./scripts/seed-demo-data.sh
```

Sign in with `demo@jobtracker.app` / `Demo123!`. See [backend/scripts/SEED-DATA.md](backend/scripts/SEED-DATA.md) for deploy and `--force` options.
